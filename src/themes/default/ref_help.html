Раздел администрирования Афиши Кино предназначен для редактирования сеансов кинотеатра.

Залы

Здесь можно создать и/или изменить список залов, где демонстрируются фильмы.

Отображется список созданых залов, каждую позицию можно "редактировать" либо "удалить", выбрав соответствющую команду в строке зала

"Редактирование" открывает страницу с формой редактирования данных, где можно поменять название зала, его описание и загрузить для него изображение.

"Залы > Добавить" : здесь Вы создаете новый зал. форма аналогичная редактированию, но все поля пусты.

Отредактировав информацию, нужно ее "Сохранить". Кнопка "очистить" возвращает содержимое полей в исходное состояние.

Фильмы

Заполняя информацию о фильмах помните что она становится общедоступной для всех кинотеатров и в этом она будет представлена посетителям портала. Поэтому старайтесь заполнять поля максимально подробно и корректно.

Поле "Статус" определяет будет ли отображаться фильм на сайте. Статус "скрыт" хорошо подойдет для хранения недооформленных фильмов. Статус "открыт" - фильм опубликован и доступен для заведениях сеансов по этому фильму всем кинотеатрам.

Создав запись о фильме, Вы можете повторно войти в режим редактирования и загрузить для фильма одно или несколько изображений.

Для загрузки изображения в правой части окна кликните кнопку "Обзор" и укажите файл. Размер картинки должен быть не меньше 150 пикселей по горизонтали.
Разрешенные для загрузки форматы изображений: "jpeg", "jpg", "jpe", "gif", "png". Обратите внимание! Формат ".bmp" не поддерживается и загружаться не будет.

Процедуру можно повторить для загрузки нескольких файлов, после чего необходимо "Сохранить". Также возможно удалить загруженные ранее изображения при помощи галочек и кнопки "Сохранить".

Сеансы

Список сеансов. аналогично списку залов для каждого сеанса есть ссылка "удалить" и "редактировать". сеансом считаются все показы конкретного Фильма в конкретном Зале.

кнопка "создать сеанс" откроет пустую форму для ввода данных

Выберите фильм, если фильма нет, Вы можете его создать в разделе меню Фильмы. Если вы не видите такой пункт, отправьте запрос с информацией о фильме администратору раздела на afisha@corp.e1.ru.

Выберите зал в котором состоятся показы фильма. Поле "О цене" необязательно должно быть числом, оно может содержать произвольную текстовую подсказку о стоимости посещения сеанса. Например, "от 100 до 200р", "студентам скидка 30% " и т. д.

Поле "Статус" определяет отображается ли информация о сеансе в Афише.

Периоды показа - диапазон дат, каждому из которых соответстует список времени начала сеансов. Периодов может быть несколько. Для добавления периода нажммит кнопку "+". Кнопка "-" соответственно удалит последний созданный при помощи "+" период. отредактировав периоды при помощи кнопки-календаря либо вручную, необходимо "Сохранить" информацию.

После сохранения у каждого периода появится кнопка "Время", по нажатию которой Вы можете указать время начала сеансов данного периода. Для ввода времени сеансов перейдите в окно справа - Время, и под заголовком Добавить введите время в формате чч:мм вместо "--:--" например "14:30". после чего появится новое поле ввода времени. вы можете не заполнять новое пустое поле а просто "Сохранить".

Чтобы отредактировать/удалить уже введенное время периода сеанса, нажмите кнопку "время" у нужного периода сеанса, справа отобразится блок "Время" и список-форма с временем сеансов: можно редактировать время в полях, либо выставить галочки для удаления позиций, после чего "Сохранить" изменения кнопкой под списком в блоке "Время".

*** Ввод времени сеансов одной строкой

При добавлении времени сеансов вы можете включить / выключить "строку ввода" нажатием на кнопку [=] слева от заголовка "Добавить". Вводите в строку текст, содержащий значения времени сеансов в числовых форматах (чч:мм, чч-мм, чч.мм), разделенные пробелами либо прочими текстовыми разделителями (;,),затем нажмите кнопку [>>] один раз - Вы увидите автоматически созданные поля времени по данным строки.

Повторное нажатие кнопки [>>] добавит данные из строки без каких либо проверок, поэтому будьте внимательны при подоюных действиях. В любом случае, данные этих полей будут сохранены лишь после нажатия кнопки "Сохранить", поэтому всегда можно проигнорировать ввод неверных данных понажимав, например, кнопку Время в левой части страницы и начав ввод заново.

Будьте внимательны - не удаляйте залы, в которых заведены сеансы!
